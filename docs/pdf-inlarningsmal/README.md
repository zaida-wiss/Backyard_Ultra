# Inlärningsmål från PDF: Backendutveckling i Node.js, databaser och säkerhet

De här filerna är gjorda utifrån PDF:ens veckoplan, rubriker och lärandemål, inte utifrån de äldre markdownfilerna i `docs/`.

Varje vecka samlar vad du behöver kunna, vad du kan bygga i Backyard Ultra-projektet och kodexempel som går att skriva av. Där det hjälper visas både JavaScript och TypeScript, så att skillnaden mellan "det fungerar" och "det är typat och tryggare" blir tydlig.
